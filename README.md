# Fluxo Básico HTTP

Cliente → Request → Servidor  
Servidor → Response → Cliente

---

# Métodos HTTP

| Método | Função |
|--------|--------|
| GET | Buscar informações |
| POST | Enviar dados |
| PUT | Atualizar dados |
| DELETE | Remover dados |

---

# Exemplo de Request

```http
GET /login HTTP/1.1
Host: site.com
User-Agent: Mozilla/5.0
```

---

# Exemplo de Response

```http
HTTP/1.1 200 OK
Content-Type: text/html
```

---

# Status Codes Importantes

| Código | Significado |
|--------|-------------|
| 200 | OK |
| 301 | Redirect |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 500 | Internal Server Error |
